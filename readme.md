# pact.ar
 
SaaS de contratos digitales para Argentina. Permite a freelancers y comerciantes independientes crear, firmar y gestionar contratos de forma simple, legal y sin papel.
 
## El problema
 
En Argentina, miles de freelancers y comerciantes operan sin contratos porque el proceso es engorroso, caro o directamente desconocido. Eso los deja desprotegidos ante conflictos con clientes o proveedores.
 
## La solución
 
pact.ar simplifica la creación y firma de contratos digitales. El usuario elige un tipo de contrato, completa los datos, y lo envía a firmar por mail. Todo queda registrado con validez legal bajo el Código Civil y Comercial (art. 288) y la Ley 25.506 de Firma Digital.
 
## Stack
 
- **Frontend:** React → Vercel
- **Backend:** Node.js + Express → Railway
- **Base de datos:** PostgreSQL → Railway
